DEPLOYMENT INSTRUCTIONS
================================================================================

Backend Deployment:

1. Build:
   cd backend
   mvn clean package

2. Run:
   java -jar target/opentron-java-backend-0.1.0.jar

3. Configure (via environment):
   TRON_PORT=8000
   ENGINE_HOST=http://localhost:11434  (Ollama)
   ENGINE_API_KEY=<optional>

Frontend Deployment:

1. Development:
   cd frontend
   npm install
   npm run dev
   → Access at http://localhost:5173

2. Production:
   npm run build
   → Output in dist/

3. Run in production:
   npm run preview
   → Or serve dist/ with any static server

Docker Deployment:

Backend:
  docker build -t opentron-backend:0.1.0 backend/
  docker run -p 8000:8000 -e ENGINE_HOST=http://ollama:11434 opentron-backend:0.1.0

Desktop App (Tauri):
  cd frontend
  npm run tauri build
  → Creates platform-specific installer
