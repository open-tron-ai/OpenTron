================================================================================
INTEGRATION WITH CLI COMMANDS
================================================================================

MemoryCmd - Knowledge Management:
  tron memory list              [Lists all entries]
  tron memory search <query>    [Full-text search]
  tron memory add <text>        [Add with category/tags]
  tron memory delete <id>       [Remove entry]
  tron memory clear             [Delete all]
  tron memory export -f csv     [Export format]
  tron memory count             [Show statistics]

VaultCmd - Secrets Management:
  tron vault list               [List secret keys]
  tron vault get <key>          [Retrieve secret]
  tron vault set <key> <val>    [Store secret]
  tron vault delete <key>       [Remove secret]
  tron vault clear              [Delete all secrets]
  tron vault export             [Export vault]
  tron vault count              [Show statistics]

TelemetryCmd - Metrics & Monitoring:
  tron telemetry status         [Show status]
  tron telemetry enable         [Start collection]
  tron telemetry disable        [Stop collection]
  tron telemetry summary        [Show statistics]
  tron telemetry recent -n 10   [Last N events]
  tron telemetry export -f csv  [Export format]
  tron telemetry clear          [Delete all events]

ModelCmd - Model Management:
  tron model list               [All models]
  tron model info <id>          [Detailed specs]
  tron model search <query>     [Full-text search]
  tron model register <id> <name> [Custom model]
  tron model pull <name>        [Download]
  tron model push <name>        [Upload]
