EXTENSIONS & CUSTOMIZATION
===========================

To add a new command:

1. Create a new class extending BaseCommand:
   ```java
   public class MyCmd extends BaseCommand {
       public static void main(String[] args) throws Exception {
           new MyCmd().execute(args);
       }
       
       @Override
       public void execute(String[] args) throws Exception {
           // Implementation here
       }
       
       @Override
       public void printUsage() {
           println("Usage: tron mycmd [options]");
       }
   }
   ```

2. Add routing in Main.java:
   ```java
   case "mycmd":
       MyCmd.main(commandArgs);
       return;
   ```

3. Compile and rebuild.
