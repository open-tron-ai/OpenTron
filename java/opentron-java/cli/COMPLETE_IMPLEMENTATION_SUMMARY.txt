  $ java -jar tron-cli.jar ask "What is machine learning?"

DEPLOYMENT OPTIONS
===================

1. JAR Executable
   mvn package
   java -jar target/tron-cli-jar-with-dependencies.jar <command>

2. Standalone Script
   echo '#!/bin/bash' > tron
   echo 'java -jar tron-cli.jar "$@"' >> tron
   chmod +x tron
   ./tron <command>

3. PATH Installation
   cp target/tron-cli-jar-with-dependencies.jar /usr/local/bin/tron
   # Create wrapper script to make executable as 'tron' command

4. Docker Container
   docker run -it opentron:java tron <command>

MAINTENANCE & SUPPORT
=====================

Adding New Commands:
1. Create new class extending BaseCommand
2. Implement execute() and printUsage()
3. Add case in Main.java switch statement
4. Compile and test

Modifying Commands:
1. Edit the command class
2. Update argument parsing if needed
3. Rebuild: mvn clean compile
4. Test thoroughly

Troubleshooting:
1. Check stderr for detailed error messages
2. Use --verbose flag for debug output
3. Run 'tron doctor' for system diagnostics
4. Check ~/.OpenTron/config.toml for configuration
