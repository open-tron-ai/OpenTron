================================================================================
QUICK START
================================================================================

Installation:
  $ git clone <repo>
  $ cd opentron-java
  $ mvn clean package
  $ java -jar cli/target/tron-cli.jar --version

Basic Usage:
  $ tron ask "What is machine learning?"
  $ tron voice chat
  $ tron memory add "Learning note"
  $ tron vault set api-key secret
  $ tron telemetry summary
  $ tron model list

Advanced Usage:
  $ tron ask "Research topic" --research --save
  $ tron voice listen --timeout 30
  $ tron memory search query
  $ tron _bootstrap --verbose
  $ tron deep-research-setup --verify
