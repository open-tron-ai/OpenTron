DEPLOYMENT OPTIONS
===================

1. JAR Executable:
   java -jar tron-cli-jar-with-dependencies.jar <command> [args]

2. Wrapper Script:
   #!/bin/bash
   java -jar /path/to/tron-cli.jar "$@"

3. System Installation:
   cp jar /usr/local/bin/tron
   Create wrapper script

4. Docker Container:
   FROM openjdk:17
   COPY tron-cli.jar /app/
   ENTRYPOINT ["java", "-jar", "/app/tron-cli.jar"]

5. Maven Plugin:
   mvn exec:java -Dexec.mainClass="io.opentron.cli.Main"

USAGE EXAMPLES
==============

# Setup
$ tron init                           # Initialize
$ tron doctor                         # Verify setup

# Interaction
$ tron ask "What is AI?"              # Query
$ tron chat                           # Interactive mode

# Management
$ tron model list                     # List models
$ tron memory search "topic"          # Search memory
$ tron scheduler list                 # View tasks

# Monitoring
$ tron scan                           # Security audit
$ tron bench run -n 10                # Run benchmarks
$ tron telemetry status               # Telemetry info

# System
$ tron daemon start                   # Start daemon
$ tron gateway status                 # Check API gateway
$ tron compose up                     # Start services
