ISSUE #2: Wrong Default Engine Host
Status: ✅ FIXED

Problem:
  - Backend was defaulting to engine.host=http://localhost:8000 (itself!)
  - Should default to http://localhost:11434 (Ollama)
  - This caused forwarding to loop back to backend, resulting in 404s

Root Cause:
  - OpentronBackendApplication.java had wrong default value
  - @Value("${engine.host:http://localhost:8000}") was incorrect
  - Should be @Value("${engine.host:http://localhost:11434}")

Solution Applied:
  ✅ Changed default from :http://localhost:8000 to :http://localhost:11434
  ✅ Backend now correctly forwards to Ollama by default
  ✅ Can still override with ENGINE_HOST environment variable

Files Modified:
  - OpentronBackendApplication.java (line 47)
